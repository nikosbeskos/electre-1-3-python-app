This is an impementation of ELECTRE III algorithm with user and file input.

The project is organised in the files included in this directory and all
files are needed to run.

To run the code CHANGE THE FILE PATH of the  .xlsx file acordingly to your
local .xlsx file location. To change the file path open "load_data.py" and
replace the "file_path" variable with your file path.

Then RUN THE "main.py" FILE ONLY.

If you want to run the the projeect with random numbers generated for the
thresholds RUN THE "main_random.py" FILE ONLY.
